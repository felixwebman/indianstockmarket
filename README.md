# Felix Online Shop

## Description
Felix Online Shop is a beginner-friendly online shopping webpage where users can browse products from China.  
This project is built with **HTML, CSS, and JavaScript** and demonstrates basic web development concepts such as layout, styling, and interactivity.

---

## Features
- Product listing with name, description, price, and "Add to Cart" button  
- Simple styling using CSS  
- Button interactivity using JavaScript (alerts when a product is added)  
- Responsive layout for mobile devices
